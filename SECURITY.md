# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Please report security vulnerabilities through [GitHub Security Advisories](https://github.com/sumitroajiprabowo/llm-wiki-mcp/security/advisories/new).

**Do not** open a public issue for security vulnerabilities.

You can expect an acknowledgment within **48 hours** and a follow-up with next steps shortly after. If the vulnerability is accepted, a fix will be released as a patch version. If declined, we will explain why.
